# 🚀 NextHire AI

AI Powered Interview Preparation Platform

## 📌 About

NextHire AI is a modern interview preparation platform that I am building while learning HTML, CSS, and JavaScript.

The goal of this project is to create a complete placement preparation platform where students can practice DSA, aptitude, interview questions, resume building, and track their learning progress.

This project will be developed step by step and later upgraded using React, Node.js, Express.js, MongoDB, and AI features.
